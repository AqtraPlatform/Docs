# Calcular array

![](../../assets/images/app-development/calculate-array.png)

## Informações gerais
A etapa “Calcular Array” é projetada para realizar operações matemáticas simples em arrays. Esta etapa é frequentemente usada em conjunto com a etapa “Agrupar Por” ou para processar dados apresentados como arrays.

Esta etapa permite que você realize várias operações matemáticas, como mínimo (min), máximo (max), soma e média, em um ou mais campos em um array.

## Parâmetros
**Configurações da Etapa:**

| Campo de Configuração | Opções de Valor         | Propósito |
|-----------------------|-------------------------|-----------|
| Nome da etapa         | -                       | Nome da etapa |
| Etapa de origem       | -                       | Selecionando a etapa anterior |
| Adicionar operação    | min, max, sum, average  | O tipo de operação matemática a ser realizada em um array |

## Casos
- **Processamento matemático de arrays**: É usado para calcular indicadores estatísticos chave de um array de dados.
- **Análise e processamento de dados**: Adequado para cenários onde arrays de dados precisam ser agregados ou resumidos.

## Exceções
- **Dependência do tipo de dados no array**: Requer que os dados no array sejam adequados para realizar a operação matemática selecionada.
- **Restrições no processamento de dados complexos**: Pode não ser adequado para operações complexas que exigem análise ou processamento avançado de dados.
