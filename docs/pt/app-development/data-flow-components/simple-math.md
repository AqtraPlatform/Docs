# Matemática Simples

![](../../assets/images/app-development/simple-math.png)

## Informações gerais
A etapa “Matemática Simples” é usada para realizar operações matemáticas simples em dados de string. Esta etapa permite que você selecione dois campos do modelo de dados e aplique uma das operações matemáticas especificadas a eles, e então escreva o resultado no terceiro campo do modelo.

## Parâmetros
**Configurações da Etapa:**

| Campo de Configuração | Opções de Valor    | Propósito |
|-----------------------|--------------------|-----------|
| Nome da etapa         | -                  | Nome da etapa |
| Etapa de origem       | Multiseleção de Catálogo | Selecionando a etapa anterior |
| Adicionar Operação    | +, -, \*, /, AND, OR, XOR | Operações a serem realizadas nos campos selecionados |

## Casos
- **Processamento de Dados Matemáticos**: Usado para realizar operações matemáticas básicas, como adição, subtração, multiplicação e divisão, bem como operações lógicas.
- **Agregação e Modificação de Dados**: Adequado para alterar valores em um modelo de dados agregando ou modificando os dados de origem usando cálculos simples.

## Exceções
- **Precisão na Seleção de Campos e Operações**: Campos e operações devem ser selecionados com precisão para alcançar o resultado esperado.
