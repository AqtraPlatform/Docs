# Slider

![](../../assets/images/app-development/slider.png)

## Informações gerais
“Slider” é um componente de UI usado para exibir e personalizar um slider com imagens.

## Parâmetros
**Propriedades do componente:**

| Grupo de configurações | Campo de configuração | Opções de valor | Propósito |
|------------------------|-----------------------|------------------|-----------|
|                        | Nome                  | -                | Nome do componente de UI no sistema |
| Comum                  | Colunas               | Botão            | Adicionar um novo objeto a um contêiner |
|                        | Mostrar setas         | true, false      | Habilitar setas para alternar imagens |

## Casos
- **Galerias de Imagens**: Ótimo para criar galerias interativas ou carrosséis de imagens.
- **Apresentações Interativas**: Usado para mostrar produtos, projetos ou qualquer outro conteúdo visual.

## Exceções
- **Uso Limitado**: Não é adequado para texto ou outro conteúdo que não envolva uma visualização de slider.
- **Configurações Visuais**: Opções de personalização limitadas para elementos visuais do slider.
