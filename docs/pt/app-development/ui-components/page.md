# Página

![](../../assets/images/app-development/page.png)

## Informações gerais
“Página” é um componente de UI para criar e personalizar páginas em uma aplicação, fornecendo estrutura e organização do conteúdo.

## Parâmetros
**Propriedades do componente**

| Campo de configuração | Opções de valor | Propósito |
| --- | --- | --- |
| Nome | - | Nome do componente de UI no sistema |
| Título | - | Nome da página |

## Casos
- **Organização de Conteúdo**: Criação de páginas separadas para separar e organizar o conteúdo da aplicação.
- **Navegação**: Fornecendo navegação clara e eficiente entre diferentes seções da aplicação.

## Exceções
- **Especificidades da Aplicação**: Limitado ao uso como um elemento estrutural de uma interface, não adequado para outros propósitos.
