# Painel

![](../../assets/images/app-development/panel.png)

## Informações gerais
“Painel” é um componente de UI projetado para exibir e configurar o contêiner básico. É utilizado para posicionar e organizar elementos na aplicação.

## Parâmetros
**Propriedades do componente**

| Campo de configuração | Opções de valor | Propósito |
| --- | --- | --- |
| Nome | - | Nome do Componente de UI no sistema |

**Propriedades CSS**

| Grupo de configurações | Campo de configuração | Opções de valor | Propósito |
| --- | --- | --- | --- |
| Layout | Orientação | Horizontal, Vertical | A propriedade permite fazer alterações dependendo da orientação de exibição |
|  | Justificar-conteúdo | Nenhum, Centro, Fim, Início, Espaço ao redor, espaço entre, espaço uniformemente | A propriedade define o alinhamento ao longo do eixo principal |
|  | Alinhar itens | Nenhum, Centro, Fim, Início, Esticar | A propriedade alinhar-itens alinha todos os elementos dentro do contêiner flexível |
|  | Largura | - | Largura do componente |
|  | Altura | - | Altura do componente |
|  | Crescer | verdadeiro, falso | A propriedade determina quanto um elemento irá crescer em relação aos demais elementos flexíveis dentro do mesmo contêiner |
|  | Margem | - | A propriedade define os preenchimentos externos em todos os quatro lados do elemento |
|  | Preenchimento | - | A propriedade define os preenchimentos internos em todos os lados do elemento |
| Aparência | Raio do canto | - | A propriedade é usada para arredondar os cantos de um elemento |
|  | Espessura da borda | - | A propriedade permite definir os limites para o elemento |
| Pincel | Fundo | - | A propriedade define a cor de fundo do elemento |
|  | Pincel da borda | - | A propriedade define a cor da borda do elemento |

## Casos
- **Organização da Interface**: Usado para estruturar e posicionar elementos da interface.

## Exceções
- **Uso Especializado**: Adequado apenas para organizar elementos, não destinado a outras funções.
