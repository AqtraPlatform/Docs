# Editor de Código

![](../../assets/images/app-development/code-editor.png)

## Informações gerais
O Editor de Código é um componente de UI especializado projetado para inserir e exibir código de programa. Ele suporta várias linguagens de programação, como JavaScript, Python, etc., e fornece funcionalidade para edição fácil de código, mas o elemento não suporta um compilador.

## Parâmetros
**Propriedades do Componente:**

| Grupo de Configurações | Campo de Configuração | Opções de Valor          | Propósito |
|------------------------|----------------------|--------------------------|-----------|
| (Configurações globais) | Nome                 | -                        | Nome do Componente de UI no sistema |
| Comum                  | Desativado           | true, false              | Desativação de um elemento |
|                        | Obrigatório          | true, false              | Campo obrigatório a ser preenchido |
|                        | Tema                 | -                        | Estilo visual do editor |
|                        | Modo                 | -                        | Linguagem de programação |
|                        | Rótulo               | -                        | Descrição do campo |
|                        | Vinculação           | Multiseleção de Catálogo | Vinculação de dados |
| Eventos                | Ao valor alterado    | -                        | Evento de alteração de valor |
|                        | Ao pressionar tecla  | -                        | Evento de pressionar tecla |
|                        | Ao soltar tecla      | -                        | Evento de soltar tecla |
|                        | Ao focar             | -                        | Evento ao focar em um elemento |
| Índice da guia        | -                    | Inteiro                  | Ordem de troca de campo |

**Propriedades CSS:**

| Grupo de Configurações | Campo de Configuração | Opções de Valor          | Propósito |
|------------------------|----------------------|--------------------------|-----------|
| Layout                 | Largura              | -                        | Largura do componente |
|                        | Altura               | -                        | Altura do componente |
|                        | Crescer              | true, false              | Estiramento do componente |
|                        | Margem               | -                        | Preenchimento externo |
|                        | Preenchimento        | -                        | Preenchimento interno |
| Aparência              | Raio de Canto        | -                        | Raio dos cantos |
|                        | Espessura da Borda   | -                        | Espessura da borda |
| Pincel                 | Fundo                | -                        | Cor de fundo |
|                        | Pincel da Borda      | -                        | Cor da borda |

## Casos
- **Edição de Código**: Usado para inserir e editar código de programa em várias linguagens de programação.
- **Treinamento e Demonstração**: Usado para fins educacionais e de demonstração para mostrar exemplos de código.

## Exceções
- **Limitações de Funcionalidade**: Dependendo da implementação, pode não suportar todos os recursos de ambientes de desenvolvimento avançados.
