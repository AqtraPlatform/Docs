# Hiperlink

![](../../assets/images/app-development/hyperlink.png)

## Informações gerais
O hiperlink na Plataforma é um componente de front-end para a criação de hiperlinks. Ele permite que os usuários naveguem facilmente entre diferentes partes do aplicativo ou para recursos externos, garantindo uma navegação eficiente e intuitiva.

## Parâmetros
**Propriedades do componente**

| Grupo de configurações | Campo de configuração | Opções de valor | Propósito |
| --- | --- | --- | --- |
|  | Nome | - | Nome do Componente de UI no sistema |
| Comum | Rótulo | - | Contém o nome do contêiner de texto |
|  | Vínculo | Multiseleção de Catálogo | Contém um campo “Uri” relacionado do modelo |
|  | Valor | - | Permite especificar um valor estático para o campo, usado para especificar um link |
| Texto | Tamanho da fonte | - | A propriedade determina o tamanho da fonte |
|  | Cor | - | Um módulo em CSS que trabalha com cores, tipos de cor e transparência. |
|  | Negrito | true, false | Propriedade que permite deixar o texto em negrito |
|  | Itálico | true, false | Propriedade que permite italicizar o texto |
|  | Alinhamento do texto | Esquerda, Direita, Centro, Justificar | A propriedade é usada para definir o alinhamento horizontal do texto |

**Propriedades CSS**

| Grupo de configurações | Campo de configuração | Opções de valor | Propósito |
| --- | --- | --- | --- |
| Layout | Largura | - | Largura do componente |
|  | Altura | - | Altura do componente |
|  | Crescer | true, false | A propriedade determina quanto um elemento crescerá em relação aos outros elementos flex dentro do mesmo contêiner |
|  | Margem | - | A propriedade define os preenchimentos externos em todos os quatro lados do elemento |
|  | Preenchimento | - | A propriedade define os preenchimentos internos em todos os lados do elemento |
| Aparência | Raio de canto | - | A propriedade é usada para arredondar os cantos de um elemento |
|  | Espessura da borda | - | A propriedade permite definir os limites para o elemento |
| Pincel | Fundo | - | A propriedade define a cor de fundo do elemento |
|  | Pincel da borda | - | A propriedade define a cor da borda do elemento |

## Casos 
- **Navegação no site**: Usado para criar links para outras páginas do aplicativo ou sites externos, melhorando a interface do usuário.
- **Melhoria da experiência do usuário**: Fornece acesso fácil a recursos e informações importantes, melhorando a experiência do usuário com o aplicativo.

## Exceções
- **Limitações de funcionalidade**: Um hiperlink pode ter interoperabilidade limitada, oferecendo apenas funcionalidade básica de clique.
- **Dependência de contexto**: É importante considerar o contexto em que os links são usados para garantir uma navegação eficaz e evitar mal-entendidos.
