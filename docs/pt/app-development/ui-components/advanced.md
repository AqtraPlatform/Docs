# Avançado

O grupo "Componentes de UI Avançados" inclui um conjunto de controles mais complexos que fornecem funcionalidade avançada para interfaces de usuário interativas e dinâmicas. Esses componentes permitem lógica de dados complexa, representações gráficas e interações do usuário que exigem personalização e integração mais profundas. Usando "Componentes de UI Avançados", os desenvolvedores podem criar interfaces intuitivas e ricas em recursos que atendem aos requisitos específicos dos processos de negócios e da experiência do usuário.
