# Grade de Pivot

![](../../assets/images/app-development/pivot-grid.png)

## Informações gerais
A Grade de Pivot é um componente de interface do usuário avançado projetado para resumir, analisar e apresentar grandes quantidades de dados. Este componente permite que você analise e mapeie conjuntos de dados complexos de forma eficiente, além de fornecer ferramentas convenientes para manipulação e visualização de dados.

## Casos
- **Análise de Dados**: A Grade de Pivot é ideal para coletar, analisar e resumir dados, especialmente quando você precisa mapear rapidamente diferentes parâmetros.
- **Relatórios e Painéis**: Ótima para criar relatórios interativos e painéis onde você precisa agregar dados e apresentá-los de uma forma conveniente para análise.

## Exceções
- **Não é para Apresentação Básica de Dados**: Se você apenas deseja exibir dados em forma tabular sem processamento analítico, é melhor usar uma Grade de Dados regular.
