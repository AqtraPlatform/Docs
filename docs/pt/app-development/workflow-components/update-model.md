# Atualizar modelo

![](../../assets/images/app-development/update-model.png)

## Informações gerais
A etapa “Atualizar Modelo” em um fluxo de trabalho é usada para salvar o modelo. Esta etapa é útil em scripts onde você deseja salvar ou atualizar o modelo de dados no diretório ou catálogo do sistema. Ela permite gerenciar efetivamente os dados, garantindo sua relevância e integridade.

## Parâmetros
**Configurações da Etapa:**

| Campo de Configuração | Propósito |
|-----------------------|-----------|
| Nome da etapa         | Nome da etapa “Atualizar Modelo”  |

## Casos
- **Atualização de Referências de Dados**: Usado para atualizar informações em diretórios ou catálogos, o que é importante quando dados existentes são alterados ou novos registros são adicionados.
- **Sincronização de Dados**: Adequado para scripts onde os dados precisam ser sincronizados entre diferentes partes do sistema ou com fontes de dados externas.

## Exceções
- **Importância da Precisão dos Dados**: Ao atualizar um modelo, é fundamental garantir que os dados sejam precisos e atualizados para evitar erros e inconsistências.
