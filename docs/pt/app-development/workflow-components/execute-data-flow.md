# Executar fluxo de dados

![](../../assets/images/app-development/execute-data-flow-workflow.png)

## Informações gerais
A etapa “Executar Fluxo de Dados” dentro do fluxo de trabalho é usada para habilitar e executar um script de fluxo de dados específico.

## Parâmetros
**Configurações da Etapa:**

| Campo de Configuração | Propósito |
|-----------------------|-----------|
| Nome da etapa         | Nome da etapa “Executar fluxo de dados” |
| Fluxo de dados vinculado | Escolhendo um script de fluxo de dados para executar |

## Casos
- **Integração de Processos de Processamento de Dados**: Usado para executar Fluxos de Dados pré-definidos.

## Exceções
- **Necessidade de Ajustar o Fluxo de Dados**: É importante definir e configurar com precisão o Fluxo de Dados que será executado para garantir que eles sejam executados corretamente e de forma eficiente.
- **Gerenciamento de Dependências entre Fluxo de Trabalho e Fluxo de Dados**: A interação entre Fluxo de Trabalho e Fluxo de Dados deve ser cuidadosamente gerenciada, especialmente em termos de transferência e processamento de dados.
