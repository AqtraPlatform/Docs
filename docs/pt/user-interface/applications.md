# Menu de Aplicações

<br>

## Componentes

Esta seção contém uma lista completa de todos os componentes disponíveis no sistema. A partir daqui, você pode visualizar os detalhes de cada componente, bem como adicionar novos componentes ou multi-componentes, expandindo as capacidades da sua aplicação. Ideal para organizar e gerenciar todos os componentes do seu projeto.

![Componentes](../assets/images/user-interface/components.png)
<br>

## Módulos Python {: #python-modules }

Esta seção oferece a capacidade de adicionar e gerenciar módulos Python comuns. Esses módulos ficam disponíveis tanto no front-end no Script de Componente quanto no back-end no Script de Execução para fluxo de dados. Isso permite que você gerencie centralmente a lógica que pode ser usada em diferentes partes da aplicação. Leia mais aqui: [Usando Python](../app-development/using-python.md)

![Módulos Python](../assets/images/user-interface/python-mod.png)
<br>

## Integrações

A seção “Integrações” desempenha um papel fundamental na criação de links entre sua aplicação e serviços externos. Aqui você pode adicionar novas integrações, remover as desnecessárias e alterar as configurações das integrações atuais, o que proporciona flexibilidade e escalabilidade da funcionalidade da aplicação.

![Menu de Integrações](../assets/images/user-interface/integrations-menu.png)
<br>

## Publicação

Uma vez que você tenha terminado de configurar e preparar seus componentes, esta seção permite que você os publique para uso futuro. Você pode escolher componentes, localizações, integrações e até mesmo módulos Python para incluir em sua publicação, tornando o processo de implantação eficiente e organizado. Leia mais aqui: [Publicar Componentes](#publication)

![Publicação](../assets/images/user-interface/publication.png)
<br>

## Exportar/Importar

Esta seção fornece ferramentas para exportar e importar configurações de componentes em formato JSON. Isso é útil para mover configurações entre diferentes ambientes, como de desenvolvimento para teste ou produção, facilitando a migração e sincronização de alterações entre diferentes instâncias do Studio.

![Exportar/Importar](../assets/images/user-interface/export-import-new.png)
