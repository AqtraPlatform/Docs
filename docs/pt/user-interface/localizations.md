# Menu de Localizações

<br>

A plataforma oferece amplas capacidades para criar interfaces de usuário multilíngues, facilitando o desenvolvimento de aplicações que suportam um público global. No momento, a plataforma suporta vários idiomas, incluindo inglês, turco e russo. Isso é alcançado por meio do uso de um sistema de localização e chaves de tradução, que permite que a interface do usuário seja adaptada dinamicamente a diferentes padrões de idioma. Esses recursos proporcionam flexibilidade e escalabilidade para o desenvolvimento de aplicações multilíngues, tornando-a ideal para a implementação de projetos internacionais complexos.
<br>

## Configurando a chave de tradução para o Componente da UI

- Ao adicionar elementos de UI a um componente, você pode configurar uma ‘Chave de Tradução’ em vez de um rótulo estático.
- Exemplo: O botão “Salvar” pode ter uma chave de tradução em vez do texto “Salvar” codificado.
- Se a chave de tradução for nova, insira seu nome e adicione-a como uma nova chave.
- Se a chave já existir, selecione-a na lista suspensa.

## Fornecendo Valores de Tradução para Chaves de Tradução

1. Abra ‘Studio→Localizações→Configurações de Localização’.
2. Selecione a chave que deseja editar e clique no lápis.
3. Faça alterações nos valores de localização para os idiomas que você deseja.
4. Confirme as alterações.
