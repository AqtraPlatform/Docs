# Menu de Armazenamento de Arquivos
<br>

O armazenamento de arquivos é uma ferramenta conveniente para gerenciar arquivos usados no estúdio de low-code da plataforma. Ele fornece uma gestão flexível dos recursos de arquivos, suportando a criação, upload e uso de arquivos como parte do desenvolvimento de aplicações.
<br>

![Armazenamento de arquivos](../assets/images/user-interface/file-storage.png)
<br>

## Estrutura de pastas

- No painel esquerdo do armazenamento de arquivos, você pode criar, excluir e gerenciar uma estrutura de pastas.
- Use o botão ‘Criar Pasta’ para criar novas pastas, incluindo subpastas.
<br>

## Gerenciando arquivos

- O painel direito mostra uma lista de todos os arquivos disponíveis para uso no estúdio.
- Os arquivos podem ser enviados para um servidor, baixados para um computador local ou excluídos.
- Use o botão ‘Enviar arquivo’ para adicionar novos arquivos ao armazenamento.
<br>

## Usando o armazenamento de arquivos no estúdio

Exemplo 1:
- Um elemento ‘Arquivo’ está disponível na barra de ferramentas e pode ser arrastado para o layout da página do componente.
- O botão ‘Escolher arquivo’ aparece, permitindo o acesso a arquivos no seu disco local ou no armazenamento de arquivos.

Exemplo 2:
- Os arquivos podem ser usados nas configurações de vários elementos da interface do usuário, por exemplo, para o elemento ‘Imagem’.
- Nas configurações do elemento, você pode selecionar um arquivo do disco ou do armazenamento de arquivos para integrar à interface do usuário.
