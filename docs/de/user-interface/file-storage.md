# Dateispeicher-Menü
<br>

Der Dateispeicher ist ein praktisches Werkzeug zur Verwaltung von Dateien, die im Plattform Low-Code-Studio verwendet werden. Er bietet eine flexible Verwaltung von Dateiresourcen und unterstützt die Erstellung, den Upload und die Nutzung von Dateien im Rahmen der Anwendungsentwicklung.
<br>

![Dateispeicher](../assets/images/user-interface/file-storage.png)
<br>

## Ordnerstruktur

- Im linken Panel des Dateispeichers können Sie eine Ordnerstruktur erstellen, löschen und verwalten.
- Verwenden Sie die Schaltfläche „Ordner erstellen“, um neue Ordner, einschließlich Unterordner, zu erstellen.
<br>

## Dateien verwalten

- Das rechte Panel zeigt eine Liste aller Dateien, die im Studio zur Verfügung stehen.
- Dateien können auf einen Server hochgeladen, auf einen lokalen Computer heruntergeladen oder gelöscht werden.
- Verwenden Sie die Schaltfläche „Datei hochladen“, um neue Dateien zum Speicher hinzuzufügen.
<br>

## Verwendung des Dateispeichers im Studio

Beispiel 1:
- Ein „Datei“-Element ist in der Symbolleiste verfügbar, das auf das Layout der Komponenten-Seite gezogen werden kann.
- Die Schaltfläche „Datei auswählen“ erscheint, die den Zugriff auf Dateien auf Ihrem lokalen Laufwerk oder im Dateispeicher ermöglicht.

Beispiel 2:
- Dateien können in den Einstellungen für verschiedene Benutzeroberflächenelemente verwendet werden, zum Beispiel für das „Bild“-Element.
- In den Einstellungen des Elements können Sie eine Datei von der Festplatte oder aus dem Dateispeicher auswählen, um sie in die Benutzeroberfläche zu integrieren.
