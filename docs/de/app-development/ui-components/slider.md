# Slider

![](../../assets/images/app-development/slider.png)

## Allgemeine Informationen
„Slider“ ist eine UI-Komponente, die verwendet wird, um einen Slider mit Bildern anzuzeigen und anzupassen.

## Parameter
**Komponenten-Eigenschaften:**

| Einstellungsgruppe | Einstellungsfeld | Wertoptionen | Zweck |
|--------------------|------------------|--------------|-------|
|                    | Name             | -            | Name der UI-Komponente im System |
| Allgemein          | Spalten          | Button       | Ein neues Objekt zu einem Container hinzufügen |
|                    | Pfeile anzeigen   | true, false  | Aktivierung von Pfeilen zum Wechseln der Bilder |

## Anwendungsfälle
- **Bildgalerien**: Ideal zum Erstellen interaktiver Galerien oder Bildkarussells.
- **Interaktive Präsentationen**: Wird verwendet, um Produkte, Projekte oder andere visuelle Inhalte zu präsentieren.

## Ausnahmen
- **Eingeschränkte Nutzung**: Nicht geeignet für Texte oder andere Inhalte, die keine Slider-Ansicht erfordern.
- **Visuelle Einstellungen**: Eingeschränkte Anpassungsoptionen für visuelle Elemente des Sliders.
