# Bestätigungsstatus

![](../../assets/images/app-development/confirmation-state.png)

## Allgemeine Informationen
Der „Bestätigungsstatus“ ist eine UI-Komponente zur Anzeige eines vorkonfigurierten Genehmigungsformulars, das zur Verwaltung der Bestätigungs- oder Genehmigungsprozesse in der Anwendung verwendet wird.

## Parameter
**Komponenten Eigenschaften:**

| Einstellungsfeld | Wertoptionen | Zweck  |
| --- | --- | --- |
| Name | - | Name der UI-Komponente im System |

**CSS-Eigenschaften**

| Einstellungsgruppe | Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- | --- |
| Layout | Breite | - | Breite der Komponente |
|  | Höhe | - | Höhe der Komponente |
|  | Wachsen | true, false | Die Eigenschaft bestimmt, wie viel ein Element im Verhältnis zu den anderen Flex-Elementen im selben Container wachsen wird |
|  | Rand | - | Die Eigenschaft definiert die äußeren Abstände auf allen vier Seiten des Elements |
|  | Innenabstand | - | Die Eigenschaft legt die inneren Abstände auf allen Seiten des Elements fest |
| Erscheinungsbild | Eckenradius | - | Die Eigenschaft wird verwendet, um die Ecken eines Elements abzurunden |
|  | Randdicke | - | Die Eigenschaft ermöglicht es Ihnen, die Grenzen für das Element festzulegen |
| Pinsel | Hintergrund | - | Die Eigenschaft legt die Hintergrundfarbe des Elements fest |
|  | Randpinsel | - | Die Eigenschaft legt die Farbe des Randes des Elements fest |

## Anwendungsfälle
- **Genehmigungsformulare**: Wird verwendet, um Schnittstellen zu erstellen, bei denen die Bestätigung oder Genehmigung des Benutzers erforderlich ist.
- **Interaktive Dialoge**: Geeignet für Dialoge, die eine Bestätigung von Benutzeraktionen erfordern.

## Ausnahmen
- **Eingeschränkte Funktionalität**: Nicht geeignet für die allgemeine Dateneingabe.
