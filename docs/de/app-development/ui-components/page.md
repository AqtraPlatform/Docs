# Seite

![](../../assets/images/app-development/page.png)

## Allgemeine Informationen
„Seite“ ist eine UI-Komponente zur Erstellung und Anpassung von Seiten in einer Anwendung, die Struktur und Organisation des Inhalts bietet.

## Parameter
**Komponenten Eigenschaften**

| Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- |
| Name | - | Name der UI-Komponente im System |
| Titel | - | Seitenname |

## Anwendungsfälle
- **Inhaltsorganisation**: Erstellung separater Seiten zur Trennung und Organisation des Anwendungsinhalts.
- **Navigation**: Bereitstellung einer klaren und effizienten Navigation zwischen verschiedenen Abschnitten der Anwendung.

## Ausnahmen
- **Anwendungsspezifika**: Beschränkt auf die Verwendung als strukturelles Element einer Schnittstelle, nicht für andere Zwecke geeignet.
