# Baumansicht

![](../../assets/images/app-development/tree-view.png)

## Allgemeine Informationen
„Baumansicht“ ist eine UI-Komponente zur Anzeige und Anpassung von Daten in hierarchischer Form, die die Navigation und Präsentation komplexer Datenstrukturen erleichtert.

## Parameter
**Komponenten Eigenschaften:**

| Einstellungsgruppe | Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- | --- |
|  | Name | - | Name der UI-Komponente im System |
| Allgemein | Datenobjekt | Mehrfachauswahl aus Katalog | Enthält eine Liste aller Komponenten |
|  | Gruppieren nach | Mehrfachauswahl aus Katalog | Eigenschaft zum Gruppieren von Daten nach ausgewähltem Feld |
|  | Sortierfeld hinzufügen | Schaltfläche | Die Eigenschaft wird verwendet, um Sortierfelder hinzuzufügen |
|  | Sortierfelder | Mehrfachauswahl aus Katalog | Ermöglicht die Auswahl des Feldes, nach dem sortiert werden soll. |
| Ereignisse | Bei geladenem Datenquelle | - | Ermöglicht das Ausführen des angegebenen Skripts, wenn die Datenquelle vollständig ist |

## Anwendungsfälle
- **Visualisierung von Hierarchien**: Ideal zur Präsentation hierarchischer Daten, wie z.B. Organisationsstruktur oder Produktklassifizierung.

## Ausnahmen
- **Komplexität der Anpassung**: Erfordert sorgfältige Konfiguration, um komplexe Datenstrukturen korrekt anzuzeigen.
