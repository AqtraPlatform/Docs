# Panel

![](../../assets/images/app-development/panel.png)

## Allgemeine Informationen
„Panel“ ist eine UI-Komponente, die entwickelt wurde, um den grundlegenden Container anzuzeigen und zu konfigurieren. Sie wird verwendet, um Elemente in der Anwendung zu positionieren und zu organisieren.

## Parameter
**Komponenten Eigenschaften**

| Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- |
| Name | - | Name der UI-Komponente im System |

**CSS-Eigenschaften**

| Einstellungsgruppe | Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- | --- |
| Layout | Orientierung | Horizontal, Vertikal | Die Eigenschaft ermöglicht Änderungen je nach Anzeigeorientierung |
|  | Justify-content | Keine, Mitte, Ende, Start, Platz um, Platz zwischen, gleichmäßig verteilen | Die Eigenschaft definiert die Ausrichtung entlang der Hauptachse |
|  | Align items | Keine, Mitte, Ende, Start, Strecken | Die align-items-Eigenschaft richtet alle Elemente innerhalb des Flex-Containers aus |
|  | Breite | - | Breite der Komponente |
|  | Höhe | - | Höhe der Komponente |
|  | Wachsen | true, false | Die Eigenschaft bestimmt, wie viel ein Element im Verhältnis zu den anderen Flex-Elementen im selben Container wachsen wird |
|  | Margin | - | Die Eigenschaft definiert die äußeren Abstände auf allen vier Seiten des Elements |
|  | Padding | - | Die Eigenschaft legt die inneren Abstände auf allen Seiten des Elements fest |
| Erscheinungsbild | Eckradius | - | Die Eigenschaft wird verwendet, um die Ecken eines Elements abzurunden |
|  | Rahmenstärke | - | Die Eigenschaft ermöglicht es, die Grenzen für das Element festzulegen |
| Pinsel | Hintergrund | - | Die Eigenschaft legt die Hintergrundfarbe des Elements fest |
|  | Rahmenpinsel | - | Die Eigenschaft legt die Farbe des Rahmens des Elements fest |

## Fälle
- **Schnittstellenorganisation**: Wird verwendet, um Schnittstellenelemente zu strukturieren und zu platzieren.

## Ausnahmen
- **Spezialisierte Verwendung**: Eignet sich nur zur Organisation von Elementen, nicht für andere Funktionen.
