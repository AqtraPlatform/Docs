# Hyperlink

![](../../assets/images/app-development/hyperlink.png)

## Allgemeine Informationen
Hyperlink in der Plattform ist eine Front-End-Komponente zur Erstellung von Hyperlinks. Sie ermöglicht es den Benutzern, einfach zwischen verschiedenen Teilen der App oder zu externen Ressourcen zu navigieren, was eine effiziente und intuitive Navigation gewährleistet.

## Parameter
**Komponenten-Eigenschaften**

| Einstellungsgruppe | Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- | --- |
|  | Name | - | Name der UI-Komponente im System |
| Allgemein | Beschriftung | - | Enthält den Namen des Textcontainers |
|  | Bindung | Multiselect des Katalogs | Enthält ein verwandtes „Uri“-Feld aus dem Modell |
|  | Wert | - | Ermöglicht die Angabe eines statischen Wertes für das Feld, das zur Angabe eines Links verwendet wird |
| Text | Schriftgröße | - | Die Eigenschaft bestimmt die Schriftgröße |
|  | Farbe | - | Ein Modul in CSS, das mit Farben, Farbtypen und Transparenz arbeitet. |
|  | Fett | true, false | Eigenschaft, die es ermöglicht, Text fett zu machen |
|  | Kursiv | true, false | Eigenschaft, die es ermöglicht, den Text kursiv zu setzen |
|  | Textausrichtung | Links, Rechts, Zentrum, Blocksatz | Die Eigenschaft wird verwendet, um die horizontale Ausrichtung des Textes festzulegen |

**CSS-Eigenschaften**

| Einstellungsgruppe | Einstellungsfeld | Wertoptionen | Zweck |
| --- | --- | --- | --- |
| Layout | Breite | - | Breite der Komponente |
|  | Höhe | - | Höhe der Komponente |
|  | Wachstum | true, false | Die Eigenschaft bestimmt, wie viel ein Element im Verhältnis zu den anderen Flex-Elementen im selben Container wachsen wird |
|  | Rand | - | Die Eigenschaft definiert die äußeren Abstände auf allen vier Seiten des Elements |
|  | Innenabstand | - | Die Eigenschaft legt die inneren Abstände auf allen Seiten des Elements fest |
| Erscheinung | Eckenradius | - | Die Eigenschaft wird verwendet, um die Ecken eines Elements abzurunden |
|  | Randdicke | - | Die Eigenschaft ermöglicht es, die Grenzen für das Element festzulegen |
| Pinsel | Hintergrund | - | Die Eigenschaft legt die Hintergrundfarbe des Elements fest |
|  | Randpinsel | - | Die Eigenschaft legt die Farbe des Randes des Elements fest |

## Fälle 
- **Seiten-Navigation**: Wird verwendet, um Links zu anderen Seiten der Anwendung oder externen Websites zu erstellen, wodurch die Benutzeroberfläche verbessert wird.
- **Verbesserung der Benutzererfahrung**: Bietet einfachen Zugang zu wichtigen Ressourcen und Informationen, was die Benutzererfahrung mit der App verbessert.

## Ausnahmen
- **Funktionsbeschränkungen**: Ein Hyperlink kann eine eingeschränkte Interoperabilität aufweisen, indem er nur grundlegende Klick-für-Klick-Funktionalität bietet.
- **Kontextabhängigkeit**: Es ist wichtig, den Kontext zu berücksichtigen, in dem Links verwendet werden, um eine effektive Navigation zu gewährleisten und Missverständnisse zu vermeiden.
