# Toolbox Flows-Registerkarte

Die Toolbox Flows-Registerkarte ermöglicht es Ihnen, neue [Datenflüsse](../data-flow-components/index.md) und [Workflows](../workflow-components/index.md) hinzuzufügen. Die Registerkarte ist nach Version 0.5.x verfügbar.

Um einen neuen Datenfluss oder Workflow hinzuzufügen, klicken Sie auf das entsprechende Symbol und ziehen Sie es in den oberen Bereich des Anwendungsfensters. Ein neuer Workflow oder Datenfluss wird dann angezeigt, der wie in den neuen [Datenflüssen](../data-flow-components/index.md) und [Workflows](../workflow-components/index.md) beschrieben konfiguriert werden kann.
