# Grundlegend

Die Gruppe "Grundlegend" umfasst eine Reihe von Schnittstellenelementen, die für die grundlegenden Funktionen der Benutzerinteraktion mit der Anwendung erforderlich sind. Diese Komponenten sind Standardsteuerelemente wie Eingabefelder, Schaltflächen und andere interaktive Elemente, die die Grundlage für die Erstellung der Struktur und Form von Benutzeroberflächen bilden.
