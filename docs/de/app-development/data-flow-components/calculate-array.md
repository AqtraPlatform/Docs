# Array berechnen

![](../../assets/images/app-development/calculate-array.png)

## Allgemeine Informationen
Der Schritt „Array berechnen“ ist dafür konzipiert, einfache mathematische Operationen auf Arrays durchzuführen. Dieser Schritt wird häufig in Verbindung mit dem Schritt „Gruppieren nach“ verwendet oder um Daten zu verarbeiten, die als Arrays präsentiert werden.

Dieser Schritt ermöglicht es Ihnen, verschiedene mathematische Operationen wie Minimum (min), Maximum (max), Summe und Durchschnitt auf einem oder mehreren Feldern in einem Array durchzuführen.

## Parameter
**Schritt Einstellungen:**

| Einstellungsfeld    | Wertoptionen            | Zweck |
|---------------------|-------------------------|-------|
| Schrittname         | -                       | Name des Schrittes |
| Quellenschritt      | -                       | Auswahl des vorherigen Schrittes |
| Operation hinzufügen | min, max, sum, average  | Die Art der mathematischen Operation, die auf ein Array angewendet werden soll |

## Anwendungsfälle
- **Mathematische Verarbeitung von Arrays**: Wird verwendet, um wichtige statistische Indikatoren eines Datenarrays zu berechnen.
- **Datenanalyse und -verarbeitung**: Geeignet für Szenarien, in denen Datenarrays aggregiert oder zusammengefasst werden müssen.

## Ausnahmen
- **Abhängigkeit von der Art der Daten im Array**: Erfordert, dass die Daten im Array für die Durchführung der ausgewählten mathematischen Operation geeignet sind.
- **Einschränkungen bei der Verarbeitung komplexer Daten**: Möglicherweise nicht geeignet für komplexe Operationen, die eine fortgeschrittene Datenanalyse oder -verarbeitung erfordern.
