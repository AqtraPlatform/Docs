# Forum communautaire

<br>

Nous vous invitons à vous inscrire sur [notre forum](https://forum.aqtra.io/), où vous pouvez échanger des expériences

[Inscrivez-vous ici](https://forum.aqtra.io/)

<br>

![Forum](assets/images/general/forumscreen1.png)

<br>

## Étapes d'inscription

<br>

1. **Inscription sur le forum :** Vous devez cliquer sur le bouton S'inscrire dans le coin supérieur droit de l'écran, puis entrer votre nom d'utilisateur, votre adresse e-mail et votre mot de passe.
   <br>

2. **Vérification du compte :** Après l'inscription, vous devez vérifier votre compte. Veuillez noter que l'e-mail de confirmation peut se retrouver dans le dossier "Spam".
   <br>

3. **Participation aux discussions :** Après vérification, vous pouvez créer des discussions sur divers sujets, répondre à d'autres participants et interagir.
