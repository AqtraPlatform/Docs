# Menu de stockage de fichiers
<br>

Le stockage de fichiers est un outil pratique pour gérer les fichiers utilisés dans le studio low-code de la plateforme. Il offre une gestion flexible des ressources de fichiers, prenant en charge la création, le téléchargement et l'utilisation de fichiers dans le cadre du développement d'applications.
<br>

![Stockage de fichiers](../assets/images/user-interface/file-storage.png)
<br>

## Structure des dossiers

- Dans le panneau de gauche du stockage de fichiers, vous pouvez créer, supprimer et gérer une structure de dossiers.
- Utilisez le bouton ‘Créer un dossier’ pour créer de nouveaux dossiers, y compris des sous-dossiers.
<br>

## Gestion des fichiers

- Le panneau de droite affiche une liste de tous les fichiers disponibles pour une utilisation dans le studio.
- Les fichiers peuvent être téléchargés sur un serveur, téléchargés sur un ordinateur local ou supprimés.
- Utilisez le bouton ‘Télécharger un fichier’ pour ajouter de nouveaux fichiers au stockage.
<br>

## Utilisation du stockage de fichiers dans le studio

Exemple 1 :
- Un élément ‘Fichier’ est disponible dans la barre d'outils et peut être glissé sur la mise en page de la page du composant.
- Le bouton ‘Choisir un fichier’ apparaît, permettant d'accéder aux fichiers sur votre disque local ou dans le stockage de fichiers.

Exemple 2 :
- Les fichiers peuvent être utilisés dans les paramètres de divers éléments de l'interface utilisateur, par exemple, pour l'élément ‘Image’.
- Dans les paramètres de l'élément, vous pouvez sélectionner un fichier sur le disque ou dans le stockage de fichiers pour l'intégrer dans l'interface utilisateur.
