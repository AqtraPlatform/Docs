# Calculer un tableau

![](../../assets/images/app-development/calculate-array.png)

## Informations générales
L'étape « Calculer un tableau » est conçue pour effectuer des opérations mathématiques simples sur des tableaux. Cette étape est souvent utilisée en conjonction avec l'étape « Regrouper par » ou pour traiter des données présentées sous forme de tableaux.

Cette étape vous permet d'effectuer diverses opérations mathématiques, telles que le minimum (min), le maximum (max), la somme et la moyenne, sur un ou plusieurs champs dans un tableau.

## Paramètres
**Paramètres de l'étape :**

| Champ de paramètre  | Options de valeur       | Objectif |
|---------------------|-------------------------|----------|
| Nom de l'étape      | -                       | Nom de l'étape |
| Étape source        | -                       | Sélectionner l'étape précédente |
| Ajouter une opération| min, max, sum, average  | Le type d'opération mathématique à effectuer sur un tableau |

## Cas d'utilisation
- **Traitement mathématique des tableaux** : Il est utilisé pour calculer des indicateurs statistiques clés d'un tableau de données.
- **Analyse et traitement des données** : Convient aux scénarios où les tableaux de données doivent être agrégés ou résumés.

## Exceptions
- **Dépendance au type de données dans le tableau** : Nécessite que les données dans le tableau soient adaptées à l'opération mathématique sélectionnée.
- **Restrictions sur le traitement de données complexes** : Peut ne pas convenir aux opérations complexes nécessitant une analyse ou un traitement avancé des données.
