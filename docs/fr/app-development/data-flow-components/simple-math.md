# Mathématiques simples

![](../../assets/images/app-development/simple-math.png)

## Informations générales
L'étape "Mathématiques simples" est utilisée pour effectuer des opérations mathématiques simples sur des données de chaîne. Cette étape vous permet de sélectionner deux champs du modèle de données et d'appliquer l'une des opérations mathématiques spécifiées, puis d'écrire le résultat dans le troisième champ du modèle.

## Paramètres
**Paramètres de l'étape :**

| Champ de paramètre | Options de valeur    | Objectif |
|--------------------|----------------------|----------|
| Nom de l'étape     | -                    | Nom de l'étape |
| Étape source       | Sélection multiple du Catalogue | Sélection de l'étape précédente |
| Ajouter une opération | +, -, \*, /, AND, OR, XOR | Opérations à effectuer sur les champs sélectionnés |

## Cas
- **Traitement des données mathématiques** : Utilisé pour effectuer des opérations mathématiques de base telles que l'addition, la soustraction, la multiplication et la division, ainsi que des opérations logiques.
- **Agrégation et modification des données** : Convient pour modifier des valeurs dans un modèle de données en agrégeant ou en modifiant les données sources à l'aide de calculs simples.

## Exceptions
- **Précision de la sélection des champs et des opérations** : Les champs et les opérations doivent être sélectionnés avec précision pour obtenir le résultat attendu.
