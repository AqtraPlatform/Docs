# Mettre à jour le modèle

![](../../assets/images/app-development/update-model.png)

## Informations générales
L'étape "Mettre à jour le modèle" dans un flux de travail est utilisée pour enregistrer le modèle. Cette étape est utile dans les scripts où vous souhaitez enregistrer ou mettre à jour le modèle de données dans le répertoire ou le catalogue du système. Elle vous permet de gérer efficacement les données, en garantissant leur pertinence et leur intégrité.

## Paramètres
**Paramètres de l'étape :**

| Champ de paramètre | Objectif |
|--------------------|----------|
| Nom de l'étape     | Nom de l'étape "Mettre à jour le modèle" |

## Cas d'utilisation
- **Mise à jour des références de données** : Utilisé pour mettre à jour les informations dans les répertoires ou les catalogues, ce qui est important lorsque des données existantes sont modifiées ou que de nouveaux enregistrements sont ajoutés.
- **Synchronisation des données** : Convient aux scripts où les données doivent être synchronisées entre différentes parties du système ou avec des sources de données externes.

## Exceptions
- **Importance de l'exactitude des données** : Lors de la mise à jour d'un modèle, il est essentiel de s'assurer que les données sont précises et à jour pour éviter les erreurs et les incohérences.
