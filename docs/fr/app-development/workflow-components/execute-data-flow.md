# Exécuter le flux de données

![](../../assets/images/app-development/execute-data-flow-workflow.png)

## Informations générales
L'étape "Exécuter le flux de données" dans le flux de travail est utilisée pour activer et exécuter un script de flux de données spécifique.

## Paramètres
**Paramètres de l'étape :**

| Champ de paramètre | Objectif |
|--------------------|----------|
| Nom de l'étape     | Nom de l'étape "Exécuter le flux de données" |
| Flux de données lié | Choisir un script de flux de données à exécuter |

## Cas
- **Intégration des processus de traitement des données** : Utilisé pour exécuter des flux de données prédéfinis.

## Exceptions
- **Besoin d'affiner le flux de données** : Il est important de définir et de configurer avec précision le flux de données qui sera exécuté pour garantir qu'il fonctionne correctement et efficacement.
- **Gestion des dépendances entre le flux de travail et le flux de données** : L'interaction entre le flux de travail et le flux de données doit être soigneusement gérée, en particulier en ce qui concerne le transfert et le traitement des données.
