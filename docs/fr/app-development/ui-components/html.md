# HTML

![](../../assets/images/app-development/html.png)

## Informations générales
HTML est le composant qui vous permet d'intégrer du code HTML brut directement dans l'interface utilisateur. Il est idéal pour présenter du contenu utilisateur, y compris du texte, des images, des tableaux et d'autres éléments qui peuvent être créés et manipulés à l'aide du HTML standard.

## Paramètres
**Propriétés du composant**

| Groupe de paramètres | Champ de paramètre | Options de valeur | Objectif |
| --- | --- | --- | --- |
|  | Nom | - | Nom du composant UI dans le système |
| Commun | Html | Bouton | Contient un espace pour éditer le code Html |

**Propriétés CSS**

| Groupe de paramètres | Champ de paramètre | Options de valeur | Objectif |
| --- | --- | --- | --- |
| Mise en page | Largeur | - | Largeur du composant |
|  | Hauteur | - | Hauteur du composant |
|  | Croissance | true, false | La propriété détermine dans quelle mesure un élément va croître par rapport aux autres éléments flex dans le même conteneur |
|  | Marge | - | La propriété définit les espacements extérieurs sur les quatre côtés de l'élément |
|  | Rembourrage | - | La propriété définit les espacements intérieurs sur tous les côtés de l'élément |
| Apparence | RayonDeCoin | - | La propriété est utilisée pour arrondir les coins d'un élément |
|  | ÉpaisseurDeBordure | - | La propriété vous permet de définir les limites de l'élément |
| Brosse | Arrière-plan | - | La propriété définit la couleur de fond de l'élément |
|  | BordureBrosse | - | La propriété définit la couleur de la bordure de l'élément |

## Cas d'utilisation
- **Contenu dynamique** : Idéal pour afficher dynamiquement du contenu nécessitant un formatage HTML, comme des articles ou des blogs.
- **Interactivité** : Permet d'intégrer des éléments interactifs, tels que des formes ou des animations, directement dans l'interface utilisateur.

## Exceptions
- **Compatibilité des navigateurs** : Le code HTML doit être soigneusement vérifié pour sa compatibilité avec différents navigateurs et appareils afin d'assurer un affichage correct.
