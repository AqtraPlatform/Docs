# Panneau

![](../../assets/images/app-development/panel.png)

## Informations générales
Le “Panneau” est un composant d'interface utilisateur conçu pour afficher et configurer le conteneur de base. Il est utilisé pour positionner et organiser des éléments dans l'application.

## Paramètres
**Propriétés du composant**

| Champ de réglage | Options de valeur | Objectif |
| --- | --- | --- |
| Nom | - | Nom du composant d'interface utilisateur dans le système |

**Propriétés CSS**

| Groupe de réglages | Champ de réglage | Options de valeur | Objectif |
| --- | --- | --- | --- |
| Mise en page | Orientation | Horizontal, Vertical | La propriété permet d'apporter des modifications en fonction de l'orientation d'affichage |
|  | Justifier le contenu | Aucun, Centre, Fin, Début, Espace autour, espace entre, espace uniformément | La propriété définit l'alignement le long de l'axe principal |
|  | Aligner les éléments | Aucun, Centre, Fin, Début, Étendre | La propriété align-items aligne tous les éléments à l'intérieur du conteneur flex |
|  | Largeur | - | Largeur du composant |
|  | Hauteur | - | Hauteur du composant |
|  | Croître | vrai, faux | La propriété détermine dans quelle mesure un élément va croître par rapport aux autres éléments flex dans le même conteneur |
|  | Marge | - | La propriété définit les marges extérieures sur les quatre côtés de l'élément |
|  | Rembourrage | - | La propriété définit les rembourrages intérieurs sur tous les côtés de l'élément |
| Apparence | Rayon de coin | - | La propriété est utilisée pour arrondir les coins d'un élément |
|  | Épaisseur de bordure | - | La propriété permet de définir les limites de l'élément |
| Brosse | Arrière-plan | - | La propriété définit la couleur de fond de l'élément |
|  | Bordure de brosse | - | La propriété définit la couleur de la bordure de l'élément |

## Cas d'utilisation
- **Organisation de l'interface** : Utilisé pour structurer et placer des éléments d'interface.

## Exceptions
- **Utilisation spécialisée** : Convient uniquement à l'organisation des éléments, non destiné à d'autres fonctions.
