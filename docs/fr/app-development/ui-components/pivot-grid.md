# Grille de pivot

![](../../assets/images/app-development/pivot-grid.png)

## Informations générales
La Grille de Pivot est un composant UI avancé conçu pour résumer, analyser et présenter de grandes quantités de données. Ce composant vous permet d'analyser et de cartographier efficacement des ensembles de données complexes, et fournit également des outils pratiques pour la manipulation et la visualisation des données.

## Cas d'utilisation
- **Analyse de données** : La Grille de Pivot est idéale pour collecter, analyser et résumer des données, surtout lorsque vous devez rapidement cartographier différents paramètres.
- **Rapports et tableaux de bord** : Parfaite pour créer des rapports interactifs et des tableaux de bord où vous devez agréger des données et les présenter sous une forme pratique pour l'analyse.

## Exceptions
- **Pas pour la présentation de données de base** : Si vous souhaitez simplement afficher des données sous forme tabulaire sans traitement analytique, il est préférable d'utiliser une Grille de Données classique.
