# Basique

Le groupe "Basique" couvre un ensemble d'éléments d'interface nécessaires aux fonctions de base de l'interaction utilisateur avec l'application. Ces composants sont des contrôles standard tels que des champs de saisie, des boutons et d'autres éléments interactifs qui servent de base à la création de la structure et de la forme des interfaces utilisateur.
