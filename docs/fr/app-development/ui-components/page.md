# Page

![](../../assets/images/app-development/page.png)

## Informations générales
“Page” est un composant UI pour créer et personnaliser des pages dans une application, fournissant structure et organisation du contenu.

## Paramètres
**Propriétés du composant**

| Champ de réglage | Options de valeur | Objectif |
| --- | --- | --- |
| Nom | - | Nom du composant UI dans le système |
| Titre | - | Nom de la page |

## Cas d'utilisation
- **Organisation du contenu** : Création de pages séparées pour séparer et organiser le contenu de l'application.
- **Navigation** : Fournir une navigation claire et efficace entre les différentes sections de l'application.

## Exceptions
- **Spécificités de l'application** : Limité à une utilisation en tant qu'élément structurel d'une interface, non adapté à d'autres fins.
