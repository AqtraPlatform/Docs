# Curseur

![](../../assets/images/app-development/slider.png)

## Informations générales
“Curseur” est un composant UI utilisé pour afficher et personnaliser un curseur avec des images.

## Paramètres
**Propriétés du composant :**

| Groupe de paramètres | Champ de paramètre | Options de valeur | Objectif |
|----------------------|--------------------|-------------------|----------|
|                      | Nom                | -                 | Nom du composant UI dans le système |
| Commun               | Colonnes           | Bouton            | Ajouter un nouvel objet à un conteneur |
|                      | Afficher les flèches | vrai, faux       | Activation des flèches pour changer d'images |

## Cas d'utilisation
- **Galeries d'images** : Idéal pour créer des galeries interactives ou des carrousels d'images.
- **Présentations interactives** : Utilisé pour présenter des produits, des projets ou tout autre contenu visuel.

## Exceptions
- **Utilisation limitée** : Pas adapté pour du texte ou d'autres contenus qui ne nécessitent pas une vue de curseur.
- **Paramètres visuels** : Options de personnalisation limitées pour les éléments visuels du curseur.
