# Menú de Almacenamiento de Archivos
<br>

El almacenamiento de archivos es una herramienta conveniente para gestionar archivos utilizados en el estudio de bajo código de la plataforma. Proporciona una gestión flexible de los recursos de archivos, apoyando la creación, carga y uso de archivos como parte del desarrollo de aplicaciones.
<br>

![Almacenamiento de archivos](../assets/images/user-interface/file-storage.png)
<br>

## Estructura de carpetas

- En el panel izquierdo del almacenamiento de archivos, puedes crear, eliminar y gestionar una estructura de carpetas.
- Usa el botón ‘Crear Carpeta’ para crear nuevas carpetas, incluyendo subcarpetas.
<br>

## Gestión de archivos

- El panel derecho muestra una lista de todos los archivos disponibles para su uso en el estudio.
- Los archivos pueden ser cargados a un servidor, descargados a una computadora local o eliminados.
- Usa el botón ‘Cargar archivo’ para añadir nuevos archivos al almacenamiento.
<br>

## Uso del almacenamiento de archivos en el estudio

Ejemplo 1:
- Un elemento ‘Archivo’ está disponible en la barra de herramientas que puede ser arrastrado al diseño de la página del componente.
- Aparece el botón ‘Elegir archivo’, permitiendo el acceso a archivos en tu unidad local o almacenamiento de archivos.

Ejemplo 2:
- Los archivos pueden ser utilizados en configuraciones para varios elementos de la interfaz de usuario, por ejemplo, para el elemento ‘Imagen’.
- En la configuración del elemento, puedes seleccionar un archivo del disco o del almacenamiento de archivos para integrarlo en la interfaz de usuario.
