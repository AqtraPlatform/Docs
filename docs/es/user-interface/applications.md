# Menú de Aplicaciones

<br>

## Componentes

Esta sección contiene una lista completa de todos los componentes disponibles en el sistema. Desde aquí, puedes ver los detalles de cada componente, así como agregar nuevos componentes o multi-componentes, ampliando las capacidades de tu aplicación. Ideal para organizar y gestionar todos los componentes de tu proyecto.

![Componentes](../assets/images/user-interface/components.png)
<br>

## Módulos de Python {: #python-modules }

Esta sección ofrece la capacidad de agregar y gestionar módulos comunes de Python. Estos módulos están disponibles tanto en el front-end en el Script de Componente como en el back-end en el Script de Ejecución para el flujo de datos. Esto te permite gestionar centralmente la lógica que se puede utilizar en diferentes partes de la aplicación. Lee más aquí: [Usando Python](../app-development/using-python.md)

![Módulos de Python](../assets/images/user-interface/python-mod.png)
<br>

## Integraciones

La sección “Integraciones” juega un papel clave en la creación de enlaces entre tu aplicación y servicios externos. Aquí puedes agregar nuevas integraciones, eliminar las innecesarias y cambiar la configuración de las integraciones actuales, lo que proporciona flexibilidad y escalabilidad a la funcionalidad de la aplicación.

![Menú de Integraciones](../assets/images/user-interface/integrations-menu.png)
<br>

## Publicación

Una vez que hayas terminado de configurar y preparar tus componentes, esta sección te permite publicarlos para su uso futuro. Puedes elegir componentes, localizaciones, integraciones e incluso módulos de Python para incluir en tu publicación, haciendo que el proceso de despliegue sea eficiente y organizado. Lee más aquí: [Publicar Componentes](#publication)

![Publicación](../assets/images/user-interface/publication.png)
<br>

## Exportar/Importar

Esta sección proporciona herramientas para exportar e importar configuraciones de componentes en formato JSON. Esto es útil para mover configuraciones entre diferentes entornos, como de desarrollo a pruebas o producción, facilitando la migración y sincronización de cambios entre diferentes instancias de Studio.

![Exportar/Importar](../assets/images/user-interface/export-import-new.png)
