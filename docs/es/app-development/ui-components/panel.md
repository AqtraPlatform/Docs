# Panel

![](../../assets/images/app-development/panel.png)

## Información general
“Panel” es un componente de UI diseñado para mostrar y configurar el contenedor básico. Se utiliza para posicionar y organizar elementos en la aplicación.

## Parámetros
**Propiedades del componente**

| Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- |
| Nombre | - | Nombre del componente de UI en el sistema |

**Propiedades CSS**

| Grupo de configuraciones | Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- | --- |
| Diseño | Orientación | Horizontal, Vertical | La propiedad permite realizar cambios dependiendo de la orientación de la pantalla |
|  | Justificar-contenido | Ninguno, Centro, Fin, Inicio, Espacio alrededor, espacio entre, espacio uniformemente | La propiedad define la alineación a lo largo del eje principal |
|  | Alinear elementos | Ninguno, Centro, Fin, Inicio, Estirar | La propiedad alinea todos los elementos dentro del contenedor flex |
|  | Ancho | - | Ancho del componente |
|  | Altura | - | Altura del componente |
|  | Crecer | true, false | La propiedad determina cuánto crecerá un elemento en relación con el resto de los elementos flex dentro del mismo contenedor |
|  | Margen | - | La propiedad define los rellenos exteriores en los cuatro lados del elemento |
|  | Relleno | - | La propiedad establece los rellenos internos en todos los lados del elemento |
| Apariencia | RadioDeEsquina | - | La propiedad se utiliza para redondear las esquinas de un elemento |
|  | GrosorDelBorde | - | La propiedad permite establecer los límites para el elemento |
| Brocha | Fondo | - | La propiedad establece el color de fondo del elemento |
|  | BrochaDelBorde | - | La propiedad establece el color del borde del elemento |

## Casos
- **Organización de la Interfaz**: Se utiliza para estructurar y colocar elementos de la interfaz.

## Excepciones
- **Uso Especializado**: Adecuado solo para organizar elementos, no destinado a otras funciones.
