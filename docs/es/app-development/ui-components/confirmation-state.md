# Estado de Confirmación

![](../../assets/images/app-development/confirmation-state.png)

## Información general
El “Estado de Confirmación” es un componente de UI para mostrar un formulario de aprobación preconfigurado utilizado para gestionar los procesos de confirmación o aprobación en la aplicación.

## Parámetros
**Propiedades del componente:**

| Campo de configuración | Opciones de valor | Propósito  |
| --- | --- | --- |
| Nombre | - | Nombre del componente de UI en el sistema |

**Propiedades CSS**

| Grupo de configuraciones | Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- | --- |
| Diseño | Ancho | - | Ancho del componente |
|  | Alto | - | Alto del componente |
|  | Crecer | true, false | La propiedad determina cuánto crecerá un elemento en relación con el resto de los elementos flex dentro del mismo contenedor |
|  | Margen | - | La propiedad define los rellenos exteriores en los cuatro lados del elemento |
|  | Relleno | - | La propiedad establece los rellenos interiores en todos los lados del elemento |
| Apariencia | RadioDeEsquina | - | La propiedad se utiliza para redondear las esquinas de un elemento |
|  | GrosorDelBorde | - | La propiedad permite establecer los límites para el elemento |
| Brocha | Fondo | - | La propiedad establece el color de fondo del elemento |
|  | BrochaDelBorde | - | La propiedad establece el color del borde del elemento |

## Casos
- **Formularios de Aprobación**: Utilizados para crear interfaces donde se requiere la confirmación o aprobación del usuario.
- **Diálogos Interactivos**: Adecuados para diálogos que requieren la confirmación de acciones del usuario.

## Excepciones
- **Funcionalidad Limitada**: No es adecuado para la entrada de datos común.
