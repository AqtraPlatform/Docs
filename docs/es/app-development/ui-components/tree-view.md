# Vista de Árbol

![](../../assets/images/app-development/tree-view.png)

## Información general
“Vista de Árbol” es un componente de UI para mostrar y personalizar datos en una forma jerárquica, lo que facilita la navegación y presentación de estructuras de datos complejas.

## Parámetros
**Propiedades del componente:**

| Grupo de configuraciones | Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- | --- |
|  | Nombre | - | Nombre del componente de UI en el sistema |
| Común | Objeto de datos | Multiselección de Catálogo | Contiene una lista de todos los componentes |
|  | Agrupar por | Multiselección de Catálogo | Propiedad para agrupar datos por el campo seleccionado |
|  | Agregar campo de orden | Botón | La propiedad se utiliza para agregar campos de ordenación |
|  | Campos de orden | Multiselección de Catálogo | Permite seleccionar el campo por el cual ordenar. |
| Eventos | Al cargar la fuente de datos | - | Permite ejecutar el script especificado cuando la fuente de datos está completa |

## Casos
- **Visualización de Jerarquías**: Ideal para presentar datos jerárquicos, como la estructura organizacional o la clasificación de productos.

## Excepciones
- **Complejidad de Personalización**: Requiere una configuración cuidadosa para mostrar correctamente estructuras de datos complejas.
