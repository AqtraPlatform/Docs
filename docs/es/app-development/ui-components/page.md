# Página

![](../../assets/images/app-development/page.png)

## Información general
“Página” es un componente de UI para crear y personalizar páginas en una aplicación, proporcionando estructura y organización del contenido.

## Parámetros
**Propiedades del componente**

| Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- |
| Nombre | - | Nombre del componente de UI en el sistema |
| Título | - | Nombre de la página |

## Casos
- **Organización del contenido**: Creación de páginas separadas para separar y organizar el contenido de la aplicación.
- **Navegación**: Proporcionar una navegación clara y eficiente entre diferentes secciones de la aplicación.

## Excepciones
- **Especificidades de la aplicación**: Limitado a su uso como un elemento estructural de una interfaz, no adecuado para otros propósitos.
