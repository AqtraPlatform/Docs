# HTML

![](../../assets/images/app-development/html.png)

## Información general
HTML es el componente que te permite integrar código HTML en bruto directamente en la interfaz de usuario. Es ideal para mostrar contenido del usuario, incluyendo texto, imágenes, tablas y otros elementos que pueden ser creados y manipulados utilizando HTML estándar.

## Parámetros
**Propiedades del componente**

| Grupo de configuraciones | Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- | --- |
|  | Nombre | - | Nombre del componente de la interfaz de usuario en el sistema |
| Común | Html | Botón | Contiene espacio para editar el código Html |

**Propiedades de CSS**

| Grupo de configuraciones | Campo de configuración | Opciones de valor | Propósito |
| --- | --- | --- | --- |
| Diseño | Ancho | - | Ancho del componente |
|  | Altura | - | Altura del componente |
|  | Crecer | true, false | La propiedad determina cuánto crecerá un elemento en relación con el resto de los elementos flexibles dentro del mismo contenedor |
|  | Margen | - | La propiedad define los rellenos exteriores en los cuatro lados del elemento |
|  | Relleno | - | La propiedad establece los rellenos interiores en todos los lados del elemento |
| Apariencia | RadioDeEsquina | - | La propiedad se utiliza para redondear las esquinas de un elemento |
|  | GrosorDelBorde | - | La propiedad te permite establecer los límites para el elemento |
| Brocha | Fondo | - | La propiedad establece el color de fondo del elemento |
|  | BrochaDelBorde | - | La propiedad establece el color del borde del elemento |

## Casos de uso
- **Contenido dinámico**: Ideal para mostrar dinámicamente contenido que requiere formato HTML, como artículos o blogs.
- **Interactividad**: Permite integrar elementos interactivos, como formas o animaciones, directamente en la interfaz de usuario.

## Excepciones
- **Compatibilidad del navegador**: El código HTML debe ser cuidadosamente revisado para garantizar la compatibilidad con diferentes navegadores y dispositivos para asegurar una visualización correcta.
