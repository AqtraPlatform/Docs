# Control deslizante

![](../../assets/images/app-development/slider.png)

## Información general
“Control deslizante” es un componente de UI utilizado para mostrar y personalizar un control deslizante con imágenes.

## Parámetros
**Propiedades del componente:**

| Grupo de configuraciones | Campo de configuración | Opciones de valor | Propósito |
|--------------------------|-----------------------|-------------------|------------|
|                          | Nombre                | -                 | Nombre del componente de UI en el sistema |
| Común                    | Columnas              | Botón             | Agregar un nuevo objeto a un contenedor |
|                          | Mostrar flechas       | true, false       | Habilitar flechas para cambiar imágenes |

## Casos
- **Galerías de imágenes**: Ideal para crear galerías interactivas o carruseles de imágenes.
- **Presentaciones interactivas**: Utilizado para mostrar productos, proyectos o cualquier otro contenido visual.

## Excepciones
- **Uso limitado**: No es adecuado para texto u otro contenido que no implique una vista de control deslizante.
- **Configuraciones visuales**: Opciones de personalización limitadas para los elementos visuales del control deslizante.
