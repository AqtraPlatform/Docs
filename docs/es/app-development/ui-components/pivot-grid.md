# Cuadro dinámico

![](../../assets/images/app-development/pivot-grid.png)

## Información general
El Cuadro Dinámico es un componente de interfaz de usuario avanzado diseñado para resumir, analizar y presentar grandes cantidades de datos. Este componente te permite analizar y mapear de manera eficiente conjuntos de datos complejos, y también proporciona herramientas convenientes para la manipulación y visualización de datos.

## Casos
- **Análisis de datos**: El Cuadro Dinámico es ideal para recopilar, analizar y resumir datos, especialmente cuando necesitas mapear rápidamente diferentes parámetros.
- **Informes y Tableros**: Excelente para crear informes interactivos y tableros donde necesitas agregar datos y presentarlos en una forma conveniente para el análisis.

## Excepciones
- **No es para Presentación Básica de Datos**: Si solo deseas mostrar datos en forma tabular sin procesamiento analítico, es mejor utilizar un Cuadro de Datos regular.
