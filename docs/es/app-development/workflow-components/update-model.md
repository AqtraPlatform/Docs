# Actualizar modelo

![](../../assets/images/app-development/update-model.png)

## Información general
El paso “Actualizar Modelo” en un flujo de trabajo se utiliza para guardar el modelo. Este paso es útil en scripts donde deseas guardar o actualizar el modelo de datos en el directorio o catálogo del sistema. Permite gestionar los datos de manera efectiva, asegurando su relevancia e integridad.

## Parámetros
**Configuraciones del paso:**

| Campo de configuración | Propósito |
|------------------------|-----------|
| Nombre del paso        | Nombre del paso “Actualizar Modelo”  |

## Casos
- **Actualización de referencias de datos**: Se utiliza para actualizar información en directorios o catálogos, lo cual es importante cuando se cambian datos existentes o se añaden nuevos registros.
- **Sincronización de datos**: Adecuado para scripts donde los datos necesitan ser sincronizados entre diferentes partes del sistema o con fuentes de datos externas.

## Excepciones
- **Importancia de la precisión de los datos**: Al actualizar un modelo, es crítico asegurar que los datos sean precisos y estén actualizados para prevenir errores e inconsistencias.
