# Ejecutar flujo de datos

![](../../assets/images/app-development/execute-data-flow-workflow.png)

## Información general
El paso “Ejecutar Flujo de Datos” dentro del flujo de trabajo se utiliza para habilitar y ejecutar un script de flujo de datos específico.

## Parámetros
**Configuraciones del paso:**

| Campo de configuración | Propósito |
|------------------------|-----------|
| Nombre del paso        | Nombre del paso “Ejecutar flujo de datos” |
| Flujo de datos vinculado| Elegir un script de flujo de datos para ejecutar |

## Casos
- **Integración de Procesos de Procesamiento de Datos**: Se utiliza para ejecutar flujos de datos predefinidos.

## Excepciones
- **Necesidad de Ajustar el Flujo de Datos**: Es importante definir y configurar con precisión el flujo de datos que se ejecutará para garantizar que se ejecute correctamente y de manera eficiente.
- **Gestión de Dependencias entre Flujo de Trabajo y Flujo de Datos**: La interacción entre el Flujo de Trabajo y el Flujo de Datos debe ser gestionada cuidadosamente, especialmente en términos de transferencia y procesamiento de datos.
