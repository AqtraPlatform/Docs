# Calcular array

![](../../assets/images/app-development/calculate-array.png)

## Información general
El paso “Calcular Array” está diseñado para realizar operaciones matemáticas simples en arrays. Este paso se utiliza a menudo junto con el paso “Agrupar Por” o para procesar datos presentados como arrays.

Este paso te permite realizar varias operaciones matemáticas, como mínimo (min), máximo (max), suma y promedio, en uno o más campos en un array.

## Parámetros
**Configuraciones del paso:**

| Campo de configuración | Opciones de valor       | Propósito |
|------------------------|-------------------------|-----------|
| Nombre del paso        | -                       | Nombre del paso |
| Paso fuente            | -                       | Seleccionar el paso anterior |
| Agregar operación      | min, max, sum, average  | El tipo de operación matemática a realizar en un array |

## Casos
- **Procesamiento matemático de arrays**: Se utiliza para calcular indicadores estadísticos clave de un array de datos.
- **Análisis y procesamiento de datos**: Adecuado para escenarios donde los arrays de datos necesitan ser agregados o resumidos.

## Excepciones
- **Dependencia del tipo de datos en el array**: Requiere que los datos en el array sean adecuados para realizar la operación matemática seleccionada.
- **Restricciones en el procesamiento de datos complejos**: Puede no ser adecuado para operaciones complejas que requieren análisis o procesamiento de datos avanzados.
