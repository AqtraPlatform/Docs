# Matemáticas simples

![](../../assets/images/app-development/simple-math.png)

## Información general
El paso “Matemáticas Simples” se utiliza para realizar operaciones matemáticas simples en datos de tipo cadena. Este paso te permite seleccionar dos campos del modelo de datos y aplicar una de las operaciones matemáticas especificadas a ellos, y luego escribir el resultado en el tercer campo del modelo.

## Parámetros
**Configuraciones del paso:**

| Campo de configuración | Opciones de valor    | Propósito |
|------------------------|----------------------|------------|
| Nombre del paso        | -                    | Nombre del paso |
| Paso fuente            | Selección múltiple de Catálogo | Seleccionando el paso anterior |
| Agregar operación      | +, -, \*, /, AND, OR, XOR | Operaciones a realizar en los campos seleccionados |

## Casos
- **Procesamiento de datos matemáticos**: Se utiliza para realizar operaciones matemáticas básicas como suma, resta, multiplicación y división, así como operaciones lógicas.
- **Agregación y modificación de datos**: Adecuado para cambiar valores en un modelo de datos mediante la agregación o modificación de los datos fuente utilizando cálculos simples.

## Excepciones
- **Precisión en la selección de campos y operaciones**: Los campos y operaciones deben ser seleccionados con precisión para lograr el resultado esperado.
